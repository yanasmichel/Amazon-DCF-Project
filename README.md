# Amazon-DCF-Project
Discounted Cash Flow Analysis for Amazon 
The analysis began by defining all key inputs and manually building each schedule required for a complete valuation. This included starting with building operating model of revenue assumptions, cost structures, margins, tax rates, discount factors, capital assumptions, and share count components. All data was organized to flow seamlessly across the model.
In this DCF Analysis the fiancial information was obtained from BamSec which has the official yearly company filings and relavant reaserch was conducted through google for variables such as historical share prices.
